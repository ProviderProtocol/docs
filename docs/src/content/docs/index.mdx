## > cat ./README.md

Modern AI development requires interacting with multiple providers—**Anthropic**, **OpenAI**, **Google**, **Stability**, and more—each with distinct APIs, authentication schemes, and response formats.

**Provider Protocol** solves this fragmentation by providing a unified protocol that works everywhere.

<CardGrid stagger>
	<Card title="> providers --list" icon="rocket">
		Switch between Anthropic, OpenAI, Google, and other providers without changing your application code.
	</Card>
	<Card title="> modality --help" icon="puzzle">
		Purpose-built interfaces: `llm()`, `embedding()`, and `image()` for each modality.

	<Card title="> config --transparent" icon="open-book">

	</Card>
</CardGrid>

## > ./demo.ts

```typescript
import { llm, embedding } from '@providerprotocol/ai';
import { anthropic } from '@providerprotocol/ai/anthropic';
import { openai } from '@providerprotocol/ai/openai';

// ┌─────────────────────────────────────────────────────────────┐
// │ LLM INFERENCE                                               │
// └─────────────────────────────────────────────────────────────┘
const claude = llm({
  model: anthropic('claude-sonnet-4-20250514'),
  system: 'You are a helpful assistant.',
});
const turn = await claude.generate('What is the capital of France?');
console.log(turn.response.text);
// => "The capital of France is Paris."

// ┌─────────────────────────────────────────────────────────────┐
// │ EMBEDDINGS                                                  │
// └─────────────────────────────────────────────────────────────┘
const embedder = embedding({
  model: openai('text-embedding-3-large'),
});
const result = await embedder.embed('Hello, world!');
console.log(`Vector dimensions: ${result.vector.length}`);
// => "Vector dimensions: 3072"
```
