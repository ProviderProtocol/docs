# Anthropic Provider

Provider for Anthropic's Claude models.

## Import

```typescript
import { anthropic } from '@providerprotocol/ai/anthropic';
import type { AnthropicLLMParams } from '@providerprotocol/ai/anthropic';
```

## Usage

```typescript
import { llm } from '@providerprotocol/ai';
import { anthropic } from '@providerprotocol/ai/anthropic';

const claude = llm({
  model: anthropic('claude-sonnet-4-20250514'),
  system: 'You are a helpful assistant.',
});

const turn = await claude.generate('Hello!');
```

## Provider-Specific Parameters

```typescript
interface AnthropicLLMParams {
  /** Maximum tokens to generate (required by Anthropic API) */
  max_tokens: number;

  /** Sampling temperature (0-1) */
  temperature?: number;

  /** Top-p (nucleus) sampling */
  top_p?: number;

  /** Top-k sampling - only sample from top K tokens */
  top_k?: number;

  /** Stop sequences - generation stops when any sequence is encountered */
  stop_sequences?: string[];

  /** Request metadata */
  metadata?: {
    /** Identifier for the end user making the request */
    user_id?: string;
  };

  /** Extended thinking configuration for reasoning models */
  thinking?: {
    type: 'enabled';
    /** Token budget for thinking process */
    budget_tokens: number;
  };

  /** Service tier for request routing */
  service_tier?: 'auto' | 'standard_only';
}
```

<Aside type="note">
Unlike other providers, Anthropic requires `max_tokens` to be explicitly set.
</Aside>

## Configuration

```typescript
import { llm } from '@providerprotocol/ai';
import { anthropic } from '@providerprotocol/ai/anthropic';

const claude = llm({
  model: anthropic('claude-sonnet-4-20250514'),
  config: {
    apiKey: process.env.ANTHROPIC_API_KEY,
    baseUrl: 'https://api.anthropic.com', // default
    timeout: 60000,
  },
  params: {
    max_tokens: 4096,
    temperature: 0.7,
  },
});
```

<Aside type="tip">
If `apiKey` is not provided, the provider will look for `ANTHROPIC_API_KEY` in environment variables.
</Aside>
